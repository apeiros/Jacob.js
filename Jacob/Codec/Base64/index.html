    mixin Jacob.Codec.Base64

    Description

    Summary

    Encode/Decode base64. Also take a look at Jacob.Codec.

    Synopsis

    Jacob.Codec.Base64.encode("Some string")      // => "U29tZSBTdHJpbmc="
Jacob.Codec.Base64.decode("U29tZSBTdHJpbmc=") // => "Some String"
