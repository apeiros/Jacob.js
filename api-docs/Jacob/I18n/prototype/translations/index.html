    instance method Jacob.I18n#translations 

    Jacob.I18n#translations() → Object

    Summary

    Returns a hash containing all locales and the corresponding flattened hash of all keys and their translations in that locale. Flattened means that - no matter how deep the hashes have been when the locales were loaded - there are no nested hashes in it.

    Synopsis

    i18n.translations(); // => {"generic": {"/some/key": "value"}, "en": {}}
